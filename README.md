## Usage

- Run node index.js to start program.

## Installation

clone repo to local machine

```
cd team-profile-generator/
```

Install dependencies

```
npm install
```

```
node index.js
```

## Tests

```
cd team-profile-generator
```

to run unit tests:

```
npm run test
```
